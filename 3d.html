3D mode requires HTTP

Three.js загружается как ES module через CDN. Firefox file:// блокирует cross-origin модули из соображений безопасности.

Запусти локальный HTTP-сервер (любой):

cd F:/WorkAI/visual/ai-conversation-viz
python -m http.server 8000

или npx serve · caddy file-server · php -S localhost:8000

Открой:

http://localhost:8000/3d.html

2D режим (index.html, standalone.html) работает по file:// — там используется bundled JS без модулей.
